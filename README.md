# Unity Claude Dev Template

A template repository for Unity game development with Claude Code integration.

## What This Is

This template provides a ready-to-use Unity development environment with Claude Code support via GitHub Actions. When you mention `@claude` in issues or PRs, Claude will automatically help with your code.

## Setup

1. **Use this template** to create your repository

2. **Configure Claude Code** by running in your terminal:
   ```bash
   claude
   ```
   Then:
   ```
   /install-github-app
   ```

3. **Test it** by creating an issue and commenting:
   ```
   @claude help me get started
   ```

## Features

- Unity 6 (6000.0.31f1) development environment via Game CI Docker
- Automated Claude Code assistance on issues and PRs
- GitHub Actions workflow pre-configured
